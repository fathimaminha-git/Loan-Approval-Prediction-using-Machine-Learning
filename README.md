⭐ Loan Approval Prediction using Machine Learning

This project focuses on predicting loan approval outcomes based on applicant data using machine learning techniques. It helps understand the key factors that influence loan approval decisions and supports automated decision-making.

🛠 Tech Stack
Language: Python
Libraries: Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn
Model: Logistic Regression
📊 Project Workflow
Cleaned and prepared the dataset by handling missing values and encoding categorical features
Performed exploratory data analysis (EDA) to understand patterns in applicant data
Applied feature engineering to improve model performance
Built a Logistic Regression model using training and testing data
Evaluated the model using accuracy and classification metrics
📈 Results

The model achieved an accuracy of 90.06% on the test dataset.

The analysis also showed that factors like credit history and income have a strong influence on loan approval decisions.

🎯 Objective

To build a simple and effective machine learning model that can predict loan approval outcomes based on applicant information.
